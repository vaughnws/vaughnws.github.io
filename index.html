Thyroid Screener

This is my first attempt to use tensorflow to classify images, it is trained to classify thyroid nodules as either benign or malignant, it was trained on 6000 images.

The model used 500 epochs, with a batch size of 16 and a learning rate of 0.001. I used a publicly available dataset from harvard med and a postgresql database to manage patient names and historys, as well as file names.
